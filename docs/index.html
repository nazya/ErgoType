ErgoType

ErgoType firmware is an extremely powerful QMK alternative. It is mainly a port of keyd with added functionality. For now it is approaching a release supporting Raspberry Pi Pico, but it is already stable and fully usable - check it out!

The WebHID configuration editor can edit ErgoType configuration files directly from a Chromium-based browser.
